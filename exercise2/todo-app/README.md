# MCP Todo Server

Implements a simple todo list as an MCP server over stdio.

## Tools

- list_todos
- add_todo (title)
- complete_todo (id)
- remove_todo (id)
- clear_completed

Data persists to `todos.json` in this directory.

Run:

```
npm start
```

## Usage

To run the server:

```
npm start
```

Or directly:

```
node server.js
```
